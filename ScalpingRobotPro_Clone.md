# ⚡ Scalping Robot Pro (Clone)

โมเดลบอทที่ถอดแบบมาจาก EA ราคาแพงในท้องตลาด ออกแบบมาสำหรับทำ **M1 Scalping บนทองคำ (XAUUSD)** โดยเฉพาะ มีจุดเด่นคือการใช้ระบบ **"Selective Grid"** ในการแก้พอร์ตเมื่อผิดทาง แทนที่จะตัดขาดทุนทันที

## 📌 แนวคิดและกลยุทธ์ (Core Strategy)
- **ประเภทกลยุทธ์:** Momentum Scalping ผสม Mean Reversion
- **สินทรัพย์ที่เหมาะสม:** XAUUSD (ทองคำ)
- **จุดแข็ง:** Win rate เข้าใกล้ 100% (เพราะใช้ Grid แก้ทาง), มีระบบหลบข่าวฉุกเฉิน (ATR Spike)

## 📊 อินดิเคเตอร์ที่ใช้ (Indicators Stack)
1. **Bollinger Bands (20, 2.5):** ใช้หากรอบการแกว่งตัวระยะสั้นมาก (M1) กางขอบให้กว้าง (Dev 2.5) เพื่อหาจังหวะ Over-extended จริงๆ
2. **RSI (14):** ใช้จับจังหวะหมดแรง (Exhaustion) ของราคา
3. **ATR (14):** ใช้เป็นไม้บรรทัดวัดแรงกระชากของแท่งเทียน เพื่อทำหน้าที่เป็น **News Filter**

## ⚙️ พารามิเตอร์ที่สำคัญ (Input Parameters)
- `InpBaseLot` (0.01): หลอดเริ่มต้นของรอบการเทรด
- `InpLotMultiplier` (1.5): ตัวคูณมาร์ติงเกลสำหรับไม้ถัดไป (เช่น 0.01 -> 0.015 -> 0.02)
- `InpGridDistancePips` (30.0): ระยะห่าง *ขั้นต่ำ* ก่อนที่จะอนุญาตให้บอทเปิดไม้แก้
- `InpMaxGridLevels` (5): อั้นจำนวนไม้แก้พอร์ต ห้ามเปิดเกินนี้ป้องกันพอร์ตแตก
- `InpPauseAfterShock` (30): จำนวนนาทีที่ให้บอท "หยุดทำงาน" หลังเจอข่าวกระชากรุนแรง

## 🧠 ตรรกะการเทรด (Trading Logic)
- **Initial Entry:** ราคาทะลุขอบ BB + RSI เข้าโซน Overbought/Oversold
- **Selective Grid Logic:** 
  - หากเปิดไม้แรกแล้วราคาลากผิดทาง บอท *จะไม่โยนไม้สองทันที* แม้จะถึงระยะ `GridDistance`
  - บอทจะรอจนกว่าจะถึงระยะ **AND** กราฟทำสัญญาณ Overbought/Oversold ซ้ำอีกรอบ ถึงจะสับไกไม้แก้ (ทำให้ได้เปรียบกว่า Grid ปกติมาก)
- **Exit:** เมื่อราคาวิ่งกลับมาหาค่าเฉลี่ย บอทจะคำนวณกำไรรวบยอดของทุกไม้ (Average Price) ถ้าบวกถึง `InpTakeProfitPips` จะปิดทุกไม้ออกพร้อมกัน

## 🛡️ การบริหารความเสี่ยง (Risk Management)
- **ATR Spike News Filter:** บอทไม่อ่านปฏิทินข่าว แต่คำนวณจากความยาวของแท่งเทียน M1 ปัจจุบัน หากยาวกว่า `ATR(14)` ถึง 3 เท่า บอทจะรู้ทันทีว่า "มีข่าว" และจะเข้าโหมด Pause 30 นาที
- **Daily Target / Stop Loss:** สามารถตั้งค่ากำไร/ขาดทุนเป็นเงิน USD (เช่น ขาดทุนถึง -$150 สั่งปิดออเดอร์ทั้งหมดและเลิกเทรดของวันนั้น)
